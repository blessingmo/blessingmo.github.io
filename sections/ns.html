Network Security

Module Objectives

  • Understand the fundamentals of vulnerability assessment in various systems.
  • Understand the fundamentals of the provision of security in information networks.
  • Be aware of the various protocols and architectures used by various network systems.
  • Understand the use and benefits of various monitoring and logging tools.
  • Understand how to create and use security vulnerability and assessment tools.

Assignments

Assignments; why Cyber Security is now a global issue and why companies need to invest in Cyber Security.

The following tasks were graded:

  • Identify and explain the architecture, structure and functionality of basic components of a computer system, considering Cyber Security issues.
  • Evaluate critically the implications of the key vulnerabilities and
  • threats of software and network security and approaches to mitigate these issues.
